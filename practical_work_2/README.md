# Practical work 2

## Software required to build and run the program
- [Node.js](https://nodejs.org/en)

## Building

1. Go to "practical_work_2" directory
2. Run the following command:
```sh
npm install
npx tsc --build
```

## Launching

1. Go to "practical_work_2" directory
2. Run the following command:
```sh
node build/app.js
```

The result of the program will appear in the console.
